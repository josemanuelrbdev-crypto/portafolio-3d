# 🌐 Portafolio 3D Interactivo - Cyberpunk Experience

Este no es un portafolio convencional. Es una experiencia inmersiva en 3D desarrollada con **Three.js** que permite explorar mis proyectos de desarrollo Full Stack a través de un entorno virtual en primera persona con una estética técnica y ciberpunk.

## 🚀 Demo En Vivo
Puedes explorar la experiencia aquí: 
👉 **(https://josemanuelrbdev-crypto.github.io/portafolio-3d/index.html)**

## ✨ Características Principales

* **Entorno 3D Inmersivo:** Desarrollado íntegramente con **Three.js** y **Canvas API**.
* **Navegación en Primera Persona:** Control de cámara suave para explorar el escenario.
* **Interactividad con Raycasting:** Detección de objetos y puntero interactivo para seleccionar proyectos.
* **Diseño UI/UX Técnico:** Interfaz de usuario integrada en el entorno 3D para una experiencia coherente.
* **Optimización de Rendimiento:** Uso eficiente de geometrías y materiales para una carga rápida en web.

## 🛠️ Stack Tecnológico

* **Core:** [Three.js](https://threejs.org/) (Motor 3D).
* **Lenguaje:** JavaScript (ES6+), HTML5, CSS3.
* **Interacción:** Raycasting para colisiones y eventos de clic.
* **Despliegue:** GitHub Pages.

## 🕹️ Controles

* **Mouse:** Rotar la cámara y mirar alrededor.
* **Clic Izquierdo:** Interactuar con los elementos del escenario.
* **Teclas (W,A,S,D):** Movimiento a través del entorno.

## 📂 Estructura del Proyecto

* `index.html`: Punto de entrada principal y contenedor del canvas.
* `js/`: Lógica de la escena, carga de modelos y controladores.
* `assets/`: Modelos 3D, texturas y sonidos ambientales.

## 🔧 Instalación Local

Si deseas probar el entorno o modificarlo localmente:

1. Clona el repositorio:
   ```bash
   git clone [https://github.com/josemanuelrbdev-crypto/portafolio-3d.git](https://github.com/josemanuelrbdev-crypto/portafolio-3d.git)

   
2. Debido a las políticas de CORS de Three.js para cargar texturas/modelos, necesitas un servidor local. Puedes usar la extensión Live Server de VS Code o: npx serve .

3. Abre http://localhost:3000 en tu navegador
